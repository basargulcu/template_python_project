# How to use?

- The new project will be created on the current directory. So switch the path to the parent of the new-project-root.
- Run `cookiecutter https://github.com/basargulcu/template_python_project.git`
- You will be asked about the parameters, ie. project name

# Parameters

The parameters defined in cookiecutter.json act as default, but also list what will be changed in the template.
Current parameters are:
- project_name
